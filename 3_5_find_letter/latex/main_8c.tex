文字列から文字を検索する 

入力：
\begin{DoxyItemize}
\item 標準入力から，まず文字xが1つ与えられる
\item 続いて，空白を含まない文字列strが1つ与えられる．長さはlen以下．
\end{DoxyItemize}

出力：
\begin{DoxyItemize}
\item 文字列str中の文字xの位置を表示する． 文字列の最初の文字がxであれば0を，2番目の文字がxであれば1を， （以下同文）出力する． 2回以上xが出現する場合には，最初の位置を表示する． ただし文字xが文字列str中に存在しない場合には-\/1を表示する．
\end{DoxyItemize}
