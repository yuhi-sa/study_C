プログラミングの説明
#include<stdio.h>
main()
{
	int i;
	int a;
	
	i=3;

	while(i>0)
	{
		a=i;
		while(a>0)
		{
        	printf("*");
		a--;
		}
	printf("\n");
	i--;
	}

	i=5;

	while(i>0)
	{
		a=i;
		while(a>0)
		{
        	printf("*");
		a--;
		}
	printf("\n");
	i--;
	}

}




まず、1つめのwhile文の条件をi>0として、文の最後のi--によりiから１を１回の操作ごとに引きi>0を満たす限り２つめのwhile文の操作を繰り返すようにする。
次に、a=iとし、２つめのwhile文の条件をa>0とし、この条件を満たす限り＊が表示されるようにし、１回の操作ごとにaから１を引き、a>0を満たす限りこの操作を繰り返すようにする。
