プログラミングの説明
#include<stdio.h>
main()
{
	int n;
	int a[11];
	for(n=0; n<=10; n++)
	{
	scanf("%d",&a[n]);
	}
	
	float b=a[10]+0.0;
	
	for(n=9;n>=0;n--)
	{
	b=a[n]+1/b;
	}
	
	printf("%f\n",b);
}
まず、intでn、a[11]には整数が入ることを宣言、a[11]は配列でa[n]には11この値が入ることを宣言
for文を用いてnが10以下で0から1ずつ増やしていくという条件で,scanfを用いて入力された値をa[n]に格納する
そして、floatを用いてbを実数型で宣言
for文を用いてnが0以上で9から1ずつ減らしていくという条件で繰り返しb=a[n]+1/bを行う
最後にprintfを用いてbを表示する
