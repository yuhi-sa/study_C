入力：
\begin{DoxyItemize}
\item 標準入力の最初は演算子を表す文字（+, -\/, $\ast$, / のいずれか）
\item それに引き続く2個の実数は演算子を適用する数値
\end{DoxyItemize}

出力：
\begin{DoxyItemize}
\item 2個の数値に演算子を適用した結果の数値を出力する
\item 数値は小数点第2位まで表示する（\%.2f）．
\end{DoxyItemize}
