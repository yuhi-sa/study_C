実数の2進数表示 

入力：
\begin{DoxyItemize}
\item 標準入力に10進数の実数が1つ与えられる（255〜-\/255まで）
\end{DoxyItemize}

出力：
\begin{DoxyItemize}
\item 与えられた実数の2進数を表示する．
\item 整数部・実数部ともに2進数8桁（0で埋める）．2の補数表現は使用しない．
\item 先頭に符号をつける（負の場合のみ）
\end{DoxyItemize}
