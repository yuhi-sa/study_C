小文字を大文字に変換する 

入力：
\begin{DoxyItemize}
\item 標準入力から，文字列が与えられる
\item 文字列は空白や記号を含み，改行'\par
'で終わる
\end{DoxyItemize}

出力：
\begin{DoxyItemize}
\item アルファベット小文字だけを大文字に変換し， その他の文字には何もしない 関数を呼び出し，入力文字列に適用し，表示する．
\end{DoxyItemize}

入力例： \begin{DoxyVerb}this is a pen
\end{DoxyVerb}
 出力例： \begin{DoxyVerb}THIS IS A PEN
\end{DoxyVerb}
